# SEO Checklist

Detailed SEO criteria for the quality-check skill. Criteria vary by platform.

## Blog / Article SEO

### Title Tag
- [ ] Contains primary keyword (preferably near the front)
- [ ] Length: 50–60 characters
- [ ] Compelling to click (not just keyword-stuffed)
- [ ] Matches search intent (informational, navigational, transactional)

### Meta Description
- [ ] Present (not auto-generated)
- [ ] Length: 140–155 characters
- [ ] Includes primary keyword naturally
- [ ] Contains a call to action or clear value proposition

### Heading Structure
- [ ] One H1 per page, matches or closely relates to title tag
- [ ] H2s cover main subtopics; H3s for sub-points
- [ ] Primary and secondary keywords appear in at least 2-3 headings
- [ ] Headings are descriptive, not generic ("Introduction" is not useful)

### Body Content
- [ ] Primary keyword in first 100 words
- [ ] Keyword density: 1-2% (not over-optimized)
- [ ] Secondary and semantic keywords distributed naturally
- [ ] Content length appropriate for topic complexity and competition
- [ ] No keyword stuffing or unnatural phrasing

### Links
- [ ] Internal links to related content on same domain (2-5 for typical articles)
- [ ] External links to authoritative sources where claims are made
- [ ] All links use descriptive anchor text (not "click here")
- [ ] No broken links

### Images
- [ ] Alt text on every image, descriptive and keyword-relevant
- [ ] File names are descriptive (not "image001.jpg")
- [ ] Images compressed for page speed

### URL
- [ ] Short and descriptive
- [ ] Contains primary keyword
- [ ] No stop words (a, the, and, etc.) unless necessary
- [ ] Hyphens between words (not underscores)

## YouTube SEO

### Title
- [ ] Primary keyword in first 5 words
- [ ] Length: 60–70 characters
- [ ] Compelling hook (number, question, or bold claim)

### Description
- [ ] Primary keyword in first 2 sentences
- [ ] Minimum 200 words
- [ ] Secondary keywords distributed naturally
- [ ] Timestamps for long videos
- [ ] Links to related content and channel

### Tags
- [ ] Primary keyword as first tag
- [ ] 5-10 specific and relevant tags (not broad generic terms)
- [ ] Mix of exact-match and broad-match variants

### Thumbnail
- [ ] Clear text readable at small sizes
- [ ] Faces/expressions when relevant (drive CTR)
- [ ] High contrast and brand consistency

## Newsletter SEO (Discoverability)

Newsletter discoverability relies more on referral and word-of-mouth than search. Key signals:

- [ ] Subject line: Clear value prop, 30-50 characters, no spam trigger words
- [ ] Preview text: First 85-140 characters show in inbox, make them count
- [ ] Consistent publish schedule (algorithm and audience expectation alignment)
- [ ] Content syndication strategy (Substack Notes, Twitter/X, LinkedIn)
