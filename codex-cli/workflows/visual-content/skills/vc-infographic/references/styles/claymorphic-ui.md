# claymorphic-ui

3D polymer clay UI with soft rounded panels and floating elements

## Color Palette

- Primary: Soft pastels - lavender (#C9B1FF), mint (#A8E6CF), peach (#FFD3B6)
- Background: Light gradient, frosted glass overlays
- Accents: Glowing soft neon, candy-tone highlights

## Visual Elements

- Smooth polymer clay textures, rounded edges
- Floating bubbles, 3D icons, mini diorama scenes
- UI cards with raised clay panels, soft depth
- Progress bars, toggle switches, pill buttons
- Cute chibi characters as mascots
- Isometric perspective elements
- Soft ambient occlusion shadows

## Typography

- Embossed clay-style text on raised panels
- Rounded sans-serif, soft glowing outlines
- Keywords on 3D button or badge elements
- Clean integration into clay panel surfaces

## Prompt Notes

Use a 3D claymorphic UI style. Smooth polymer clay textures, rounded edges, soft UI. Think: Blender 3D / pastel clay / cute isometric elements. Use expressive 3D elements: floating bubbles, 3D icons, mini diorama scenes, cute chibi characters. Highlight keywords with embossed text, raised clay panels, soft glowing buttons. Organize with clean UI layout: title with floating 3D props, feature cards with 3D icons, progress bars or toggle switches. No flat 2D, no photorealism, no harsh shadows, no metallic textures. Keep soft breathable margins.

## Best For

Tech product introductions, feature showcases, app tutorials, AI tool guides, playful knowledge cards, trendy social content
