# claymorphic-ui

3D polymer clay UI with soft rounded panels and floating elements

## Design Aesthetic

Smooth 3D claymorphic UI style with polymer clay textures and rounded edges. Blender 3D / pastel clay / cute isometric aesthetic. Clean UI layout with raised panels, floating 3D props, and soft depth. Expressive elements like bubbles, 3D icons, mini diorama scenes, and cute chibi characters enhance memorability.

## Background

- Color: Soft pastel gradients (#F0E6FF to #E8F8F0)
- Texture: Frosted glass overlays, soft ambient light

## Visual Elements

- Smooth polymer clay textures, rounded edges
- Floating bubbles, 3D icons, mini diorama scenes
- UI cards with raised clay panels, soft depth
- Progress bars, toggle switches, pill buttons
- Cute chibi characters as mascots
- Isometric perspective elements
- Soft ambient occlusion shadows

## Style Rules

### Do

- Use 3D clay textures on all elements
- Keep rounded, soft edges everywhere
- Add floating decorative 3D props
- Use soft shadows with ambient occlusion
- Create clear horizontal card sections

### Don't

- Use flat 2D or photorealistic imagery
- Apply harsh, sharp, or metallic textures
- Overcrowd layout with dense text
- Use hard drop shadows or outlines
- Create cluttered compositions

## Best For

Tech product infographics, feature showcase scenes, app tutorials, AI tool comparisons, trendy knowledge frameworks, modern social content
