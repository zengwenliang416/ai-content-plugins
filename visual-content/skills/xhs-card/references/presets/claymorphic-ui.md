# Claymorphic UI Style

3D polymer clay UI with soft rounded panels, floating elements, and cute chibi characters.

## Element Combination

```yaml
canvas:
  ratio: portrait-3-4
  grid: single | dual

image_effects:
  cutout: none
  stroke: none
  filter: soft-glow | clay-depth

typography:
  decorated: embossed | badge
  tags: pill-3d | button-3d
  direction: horizontal

decorations:
  emphasis: glow-soft | raised-panel
  background: gradient-pastel | frosted-glass
  doodles: bubbles | 3d-icons | chibi-characters
  frames: clay-card | rounded-panel
```

## Color Palette

| Role | Colors | Hex |
|------|--------|-----|
| Primary | Lavender, mint, peach, sky blue | #C9B1FF, #A8E6CF, #FFD3B6, #87CEEB |
| Background | Soft gradient, frosted white | #F0E6FF, #E8F8F0 |
| Accents | Candy pink, soft neon green | #FF9ED2, #7DFFB3 |

## Visual Elements

- Smooth polymer clay textures, rounded edges
- Floating bubbles, 3D icons, mini diorama scenes
- Raised clay panels with soft depth shadows
- Progress bars, toggle switches, pill buttons
- Cute chibi characters as mascots
- Isometric perspective elements

## Typography

- Embossed text on raised clay panels
- Rounded sans-serif with soft glow outlines
- Keywords as 3D button or badge elements
- Clean integration into panel surfaces

## Best Layout Pairings

| Layout | Compatibility | Use Case |
|--------|---------------|----------|
